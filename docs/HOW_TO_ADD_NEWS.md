# 📰 Как добавить новость или афишу

---

## 🎵 Афиша (концерты, события)

### Быстрый способ

1. Положи файлы в `public/new_for_sort/`:
   - **Фото**: `EventName.jpg`
   - **Текст**: `EventName.txt` (первая строка = заголовок, дальше текст)
2. Запусти: `node scripts/add-afisha.js`

> ⚠️ Для корректного определения даты указывай в тексте: `31 марта`, `22 апреля` и т.д.
> Время: `19:00` или `🕖 19:00`. Место: `📍 Дворец культуры`.

### Ручной способ

Создай файл `src/afisha/{дата}-{slug}.md`:
```markdown
---
title: "Название мероприятия"
date: 2026-03-31
time: "19:00"
venue: Концертный зал
cover: /afisha/images/2026/2026-03-31-slug/cover.jpg
tags: []
---

Описание мероприятия...
```

## 📝 Новости

### Шаг 1: Положи файлы в `public/new_for_sort/`
- **Фото**: `NewsName_1.jpg`, `NewsName_2.jpg`, `NewsName_3.jpg`...
- **Текст**: `NewsName.txt` (первая строка = заголовок, остальное = текст)

### Шаг 2: Запусти скрипт обработки
```bash
node scripts/add-news.js
```

Скрипт автоматически:
- ✅ Создаст папку для изображений в `public/news/images/{год}/{slug}/`
- ✅ Переименует первое фото в `cover.jpg`, остальные в `gallery-1.jpg`, `gallery-2.jpg`...
- ✅ Создаст `.md` файл в `src/news/{год}/`
- ✅ Удалит обработанные файлы из `new_for_sort/`

---

## Ручной способ (полный контроль)

### 1. Создай папку для фото
```
public/news/images/{год}/{дата}-{короткий-slug}/
```
Пример: `public/news/images/2026/2026-04-15-vesenniy-koncert/`

### 2. Скопируй фото
- Первое фото → `cover.jpg` (обложка)
- Остальные → `gallery-1.jpg`, `gallery-2.jpg`...

### 3. Создай `.md` файл
Путь: `src/news/{год}/{дата}-{slug}.md`

```markdown
---
title: "Заголовок новости"
date: 2026-04-15
category: konzert
cover: /news/images/2026/2026-04-15-vesenniy-koncert/cover.jpg
tags: []
---

Текст новости...

<!-- gallery -->
- /news/images/2026/2026-04-15-vesenniy-koncert/gallery-1.jpg
- /news/images/2026/2026-04-15-vesenniy-koncert/gallery-2.jpg
```

### 4. Категории
| Значение | Иконка | Описание |
|----------|--------|----------|
| `konzert` | 🎵 | Концерт |
| `konkurs` | 🏆 | Конкурс/фестиваль |
| `masterclass` | 🎓 | Мастер-класс |
| `announcement` | 📢 | Объявление |

## Проверка
После добавления новости:
1. Запусти `npm run dev` (если не запущен)
2. Открой http://localhost:5173
3. Прокрути до блока "Новости"

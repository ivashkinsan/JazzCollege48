# Логотип эстрадного отделения

## Обзор

Логотип эстрадного отделения ЛОКИ им. К.Н. Игумнова разработан на основе визуального стиля сайта и отражает музыкальную направленность отделения.

```

```

## Варианты логотипа

### 1. Основной круглый логотип

**Файл:** `logo-main.svg`

**Размер:** 300×300 px

**Использование:**
- Главная страница (hero-секция, about)
- Печатные материалы (афиши, буклеты)
- Социальные сети (аватар)
- Мероприятия и концерты

**Элементы:**
- Стилизованная музыкальная нота (восьмая) в центре
- Золотой градиент (#E8C85A → #D4AF37 → #B8942E)
- Фиолетовые акценты (#a855f7 → #7e14ff) — в стиле favicon сайта
- Абстрактные световые лучи — отсылка к дизайну favicon
- Анимированные звуковые волны (SVG animation)
- Текст по кругу: "ЭСТРАДНОЕ ОТДЕЛЕНИЕ"
- Нижний текст: "ЛОКИ им. К.Н. ИГУМНОВА"
- Год основания: "С 1981"
- Декоративные звёзды/блики

**Особенности:**
- Радиальный градиентный фон (тёмно-фиолетовый → чёрный)
- SVG-фильтры для эффекта свечения (glow)
- CSS-анимация звуковых волн

## Цветовая схема

### Градиенты

| Название | Цвета | HEX |
|----------|-------|-----|
| **Золотой** | Светлый → Основной → Тёмный | `#E8C85A` → `#D4AF37` → `#B8942E` |
| **Фиолетовый** | Светлый → Основной | `#a855f7` → `#7e14ff` |

### Основные цвета

| Элемент | Цвет | HEX | RGB |
|---------|------|-----|-----|
| Золотой акцент | Accent | `#D4AF37` | rgb(212, 175, 55) |
| Золотой светлый | Accent Light | `#E8C85A` | rgb(232, 200, 90) |
| Фиолетовый | Purple | `#7e14ff` | rgb(126, 20, 255) |
| Фиолетовый светлый | Purple Light | `#a855f7` | rgb(168, 85, 247) |
| Фон тёмный | Background | `#0a0a0a` | rgb(10, 10, 10) |
| Фон фиолетовый | BG Purple | `#1a1a2e` | rgb(26, 26, 46) |
| Текст вторичный | Text Secondary | `#b0b0b0` | rgb(176, 176, 176) |

| Голубой акцент | Blue Accent | `#47bfff` | rgb(71, 191, 255) |

## SVG-фильтры и эффекты

### Эффект свечения (Glow)

Используется в логотипе для создания мягкого свечения:

```svg
<filter id="glow" x="-50%" y="-50%" width="200%" height="200%">
  <feGaussianBlur stdDeviation="3" result="coloredBlur"/>
  <feMerge>
    <feMergeNode in="coloredBlur"/>
    <feMergeNode in="SourceGraphic"/>
  </feMerge>
</filter>
```

**Использование:**
```svg
<g filter="url(#glow)">
  <!-- Элементы ноты -->
</g>
```

### Градиенты

**Золотой градиент:**
```svg
<linearGradient id="goldGradient" x1="0%" y1="0%" x2="100%" y2="100%">
  <stop offset="0%" stop-color="#E8C85A"/>
  <stop offset="50%" stop-color="#D4AF37"/>
  <stop offset="100%" stop-color="#B8942E"/>
</linearGradient>
```

**Фиолетовый градиент:**
```svg
<linearGradient id="purpleGradient" x1="0%" y1="0%" x2="100%" y2="100%">
  <stop offset="0%" stop-color="#a855f7"/>
  <stop offset="100%" stop-color="#7e14ff"/>
</linearGradient>
```

## Технические требования

### Форматы файлов

| Формат | Использование |
|--------|---------------|
| SVG | Веб-сайт, печать (вектор) |
| PNG | Социальные сети, презентации |
| PDF | Печатные материалы |

### Разрешение для растра

- **Веб:** 72 DPI
- **Печать:** 300 DPI
- **Retina:** 144 DPI

### Прозрачность

Логотип должен иметь прозрачный фон для использования на цветных фонах.

### Экспорт из SVG

```bash
# Конвертация в PNG (Inkscape)
inkscape --export-type=png --export-width=300 logo-main.svg

# Конвертация в PNG (ImageMagick)
convert -density 300 logo-main.svg -resize 300x300 logo-main.png
```

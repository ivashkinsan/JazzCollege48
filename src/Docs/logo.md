## Варианты логотипа

### 1. Основной круглый логотип

**Файл:** `logo-main.svg` | **Размер:** 300×300 px

**Использование:**
- Главная страница (hero-секция, about)
- Печатные материалы (афиши, буклеты)
- Социальные сети (аватар)

**Элементы:**
- Стилизованная музыкальная нота (восьмая) в центре
- Золотой градиент (#E8C85A → #D4AF37 → #B8942E)
- Фиолетовые акценты (#a855f7 → #7e14ff)
- Текст по кругу: "ЭСТРАДНОЕ ОТДЕЛЕНИЕ"
- Нижний текст: "ЛОКИ им. К.Н. ИГУМНОВА"
- Год основания: "С 1981"

---

### 2. Горизонтальная версия

**Файл:** `logo-horizontal.svg` | **Размер:** 400×80 px

**Использование:**
- Шапка сайта (Header)
- Электронные письма
- Документация и бланки
- Визитки

**Элементы:**
- Компактная круглая иконка с нотой (слева)
- Вертикальный золотой разделитель
- Название отделения (золотой градиент)
- Подзаголовок с названием колледжа (серый)

---

### 3. Favicon

**Файл:** `favicon.svg` | **Размер:** 64×64 px

**Использование:**
- Иконка вкладки браузера
- Закладки
- PWA иконка

## Цветовая схема

### Градиенты

| Название | Цвета | HEX |
|----------|-------|-----|
| **Золотой** | Светлый → Основной → Тёмный | `#E8C85A` → `#D4AF37` → `#B8942E` |
| **Фиолетовый** | Светлый → Основной | `#a855f7` → `#7e14ff` |

### Основные цвета

| Элемент | HEX | RGB |
|---------|-----|-----|
| Золотой акцент | `#D4AF37` | rgb(212, 175, 55) |
| Золотой светлый | `#E8C85A` | rgb(232, 200, 90) |
| Фиолетовый | `#7e14ff` | rgb(126, 20, 255) |
| Фиолетовый светлый | `#a855f7` | rgb(168, 85, 247) |
| Фон тёмный | `#0a0a0a` | rgb(10, 10, 10) |
| Текст вторичный | `#b0b0b0` | rgb(176, 176, 176) |
| Голубой акцент | `#47bfff` | rgb(71, 191, 255) |

### Связь с дизайн-системой

Логотип использует те же CSS-переменные, что и сайт:

```css
:root {
  --color-accent: #D4AF37;
  --color-accent-hover: #E8C85A;
  --color-accent-dark: #B8942E;
  --color-bg: #0a0a0a;
}
```

## Охранная зона

Минимальные отступы вокруг логотипа:

```
     30px
  ┌───────────┐
30│   LOGO    │30
  │           │
  └───────────┘
     30px
```

**Правило:** Минимальный отступ равен 1/5 высоты логотипа.

## Минимальный размер

### Для печати
- **Круглый:** не менее 40 мм в диаметре
- **Горизонтальный:** не менее 120 мм в ширину

### Для экрана
- **Круглый:** не менее 80×80 px
- **Горизонтальный:** не менее 200×40 px
- **Favicon:** 16×16, 32×32, 64×64 px

## Технические требования

### Форматы файлов

| Формат | Использование |
|--------|---------------|
| SVG | Веб-сайт, печать (вектор) |
| PNG | Социальные сети, презентации |
| PDF | Печатные материалы |

### Разрешение для растра

- **Веб:** 72 DPI
- **Печать:** 300 DPI
- **Retina:** 144 DPI

### Прозрачность

Логотип должен иметь прозрачный фон для использования на цветных фонах.

### Экспорт из SVG

```bash
# Конвертация в PNG (Inkscape)
inkscape --export-type=png --export-width=300 logo-main.svg

# Конвертация в PNG (ImageMagick)
convert -density 300 logo-main.svg -resize 300x300 logo-main.png
```

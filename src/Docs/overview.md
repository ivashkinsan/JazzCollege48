# Обзор проекта

## Назначение проекта

Веб-сайт эстрадного отделения Липецкого областного колледжа искусств им. К.Н. Игумнова — официальный информационный ресурс, предназначенный для абитуриентов, студентов, преподавателей и всех интересующихся эстрадным образованием в регионе.

## Цели и задачи

### Основные цели

1. **Информирование абитуриентов** — предоставление полной информации о специальности, условиях поступления и обучении
2. **Презентация отделения** — демонстрация достижений, преподавательского состава и творческих коллективов
3. **Коммуникация** — публикация новостей, афиши концертов и мероприятий
4. **Имиджевая функция** — формирование современного образа учебного заведения

### Задачи

- Предоставить структурированную информацию о специальности 53.02.02
- Показать материально-техническую базу отделения
- Продемонстрировать достижения студентов и выпускников
- Обеспечить удобный доступ к контактной информации

## О эстрадном отделении

### История

Эстрадное отделение ЛОКИ им. К.Н. Игумнова было основано в **1981 году** и является одним из ведущих подразделений колледжа.

### Образовательная программа

**Специальность:** 53.02.02 "Музыкальное искусство эстрады (по видам)"

**Уровень:** Программа углубленной подготовки

**Квалификация выпускника:**
- Артист
- Преподаватель
- Руководитель эстрадного коллектива

**Срок обучения:** 3 года 10 месяцев

**Форма обучения:** Очная

**Общая трудоёмкость:** 7722 часа

### Профили подготовки

1. **Инструменты эстрадного оркестра**
   - Ударные инструменты
   - Духовые инструменты
   - Струнные инструменты
   - Клавишные инструменты

2. **Эстрадное пение**

### Инструментальная база

Отделение готовит музыкантов по следующим направлениям:

| Категория | Инструменты |
|-----------|-------------|
| Клавишные | Фортепиано, синтезаторы, электроорган |
| Духовые | Саксофон, труба, тромбон |
| Ударные | Ударная установка, перкуссия |
| Струнные | Электрогитара, бас-гитара, контрабас |

## Целевая аудитория

1. **Абитуриенты** — выпускники школ, планирующие поступление
2. **Родители абитуриентов** — лица, принимающие решения об образовании
3. **Студенты** — текущие обучающиеся отделения
4. **Преподаватели** — сотрудники колледжа
5. **Партнёры** — концертные организации, филармонии
6. **Выпускники** — бывшие студенты отделения

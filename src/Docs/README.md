## Краткое описание

Проект — современный одностраничный сайт (SPA) эстрадного отделения ЛОКИ им. К.Н. Игумнова на базе **React 19** с **TypeScript** и сборщиком **Vite**.

### Основные возможности

- 🎓 Информация о специальности 53.02.02 "Музыкальное искусство эстрады"
- 👨‍🏫 Каталог преподавателей отделения
- 🎵 Информация об ансамблях и коллективах
- 🏆 Достижения студентов и выпускников
- 📅 Афиша концертов и мероприятий
- 📰 Новости отделения
- 📋 Информация для абитуриентов

### Технические характеристики

- **Фреймворк:** React 19.2.4
- **Язык:** TypeScript 6.0.2
- **Сборщик:** Vite 8.0.1
- **Стилизация:** CSS с CSS-переменными
- **Темизация:** Тёмная тема с золотыми акцентами

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

# Сборка production-версии
npm run build

# Предварительный просмотр сборки
npm run preview

# Проверка типов
npm run type-check

# Линтинг
npm run lint
```

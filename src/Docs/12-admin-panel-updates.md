# Последние изменения в админ-панели (июль 2026)

## 1. Админ-панель: Предпросмотр (PreviewPane)
Реализован полноценный предпросмотр для всех вкладок (Новости, Афиши, Достижения, Выпускники, Видео, Библиотека, Фотоальбом).

*   **Техническое решение:** Вместо использования компонентов фронтенда напрямую (что вызывало проблемы с CSS-модулями и хешированием классов), перешли на использование глобальных CSS-классов.
*   **Стилизация:**
    *   Созданы файлы стилей: `src/admin/admin-preview.css` (для выпускников) и `src/admin/admin-preview-news.css` (новости, достижения, афиши, видео), `src/admin/admin-preview-others.css` (доп. стили).
    *   Стили в этих файлах являются глобальными, что обеспечивает их корректное отображение в `PreviewPane` админ-панели.
    *   Для карточки «Достижения» задано соотношение сторон формата A4 (`1 / 1.414`) для корректного отображения дипломов/сертификатов.
    *   Изображения в предпросмотрах ограничены по размеру для удобства редактирования.

## 2. Изменения в БД и Backend (admin-server.ts)
*   **Миграция схемы БД:** Колонка `position` в таблице `graduates` переименована в `instrument`.
*   **Логика сохранения:** В `admin-server.ts` заменены операции `fs.rename` на комбинацию `fs.copyFile` + `fs.unlink` для всех операций с файлами. Это устранило ошибку `EPERM` (operation not permitted) при перемещении файлов на Windows.
*   **API:** Обновлены API-эндпоинты для корректной обработки поля `instrument`.

## 3. Админ-панель: Форма редактирования «Выпускники»
*   Обновлены поля формы:
    *   «Должность» заменена на «Инструмент».
    *   «Выдающийся выпускник» переименован в «Наша гордость» (checkbox).
    *   Добавлены поля: «Биография», «Место работы/учебы».
*   Обновлена таблица в админке: отображаются актуальные колонки («Инструмент», «Место работы/учебы», «Биография» с превью).
*   В форме редактирования теперь корректно отображается полный путь к файлу (например, `/vipuskniki/filename.jpg`), который будет записан в БД.
